# rl-sac-trading
Hierarchical SAC Framework for XAU/USD Trading
This repository presents a hierarchical reinforcement learning framework using Soft Actor-Critic (SAC) for adaptive, risk-conscious XAU/USD trading. Addressing the limitations of traditional trading and on-policy RL methods (e.g., PPO, A3C), which struggle with non-stationary market dynamics and dynamic risk management, we propose a low-level SAC agent trained on 1-hour XAU/USD data (2010-2020) with Gramian Angular Field heatmaps, volatility indicators (ATR), and economic event flags to optimize trading actions (position size, leverage). A meta-controller detects market regime shifts (trending, volatile, flat) using meta-states like volatility clustering or CUSUM tests, selecting sub-policies to guide the SAC agent. The framework is retrained on 2020-2025 data for online adaptation and backtested to evaluate Sharpe ratio and maximum drawdown against benchmarks (moving average crossovers, DQN, PPO), offering a scalable solution for volatile assets.
